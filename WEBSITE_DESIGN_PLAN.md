# LOINC.org.tw 網站設計方案

> **設計定位**：專業學術型醫療資訊平台
> **設計目標**：建立台灣醫學資訊學會 (TAMI) 維護的權威 LOINC 資訊發布平台

---

## 🎯 設計理念

### 核心價值
- **專業性**：展現學術機構的嚴謹與權威
- **可信度**：醫療標準組織的正式形象
- **易讀性**：清晰的資訊層級，便於快速獲取資訊
- **國際觀**：符合國際醫療標準組織的視覺風格

### 參考標準
- HL7 International (https://www.hl7.org/)
- LOINC Official (https://loinc.org/)
- WHO (https://www.who.int/)
- NIH (https://www.nih.gov/)

## 🔧 技術實現

### 1. WordPress 主題選擇

**推薦主題**：
- **Twenty Twenty-Four**（免費，區塊主題）
- **Astra**（免費/付費，高度自定義）
- **GeneratePress**（輕量快速）

**或自定義子主題**：
- 基於 Twenty Twenty-Four
- 加入自定義 CSS
- 修改模板文件

### 2. 必要插件

- **Elementor / Gutenberg**：頁面編輯器
- **Yoast SEO**：搜索引擎優化
- **WP Rocket**：緩存加速
- **Contact Form 7**：聯絡表單

### 3. 自定義 CSS 位置

```bash
WordPress 後台 > 外觀 > 自定義 > 額外 CSS
或
佈景主題 > 編輯 > style.css
```
