## 每日 CEO 訓練 Routine

### 早晨戰略時段 (6:00-8:00)
```bash
# 每日晨間 CEO 檢查清單
node check-gmail.js          # 檢查重要郵件
node check-calendar.js       # 確認今日行程
echo "今日三大戰略目標："     # 設定當日優先級
```

**思考框架**:
1. **今日最重要的決策是什麼？**
2. **哪個項目需要我的直接參與？**
3. **有什麼可以授權給團隊？**

### 中午決策時段 (12:00-13:00)
**快速決策訓練**:
- 收集上午的資訊
- 做出 2-3 個重要決定
- 發送指示給相關團隊

### 晚間反思時段 (20:00-21:00)
**CEO 反思問題**:
1. 今天創造了什麼價值？
2. 哪些決策可以更好？
3. 明天的重點是什麼？
